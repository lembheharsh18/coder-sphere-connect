# Coder Sphere Connect

An application for connecting coders.

## Project Setup

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Install dependencies
npm install
```

### Development

```sh
# Start the development server
npm run dev

# Start the backend server
npm run server

# Run both concurrently
npm run dev:all
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Express
- Prisma
- PostgreSQL

## Deployment

### Frontend (Vercel)

The frontend is deployed on Vercel. Pushing changes to the main branch will verify and redeploy the frontend.

### Backend (Render)

The backend is deployed on Render. Pushing changes to the main branch will trigger a redeploy.
